Projeto desenvolvido no **SENAI** com o objetivo de criar uma **plataforma de delivery para a Beirute Esfiharia**, unindo tecnologia, usabilidade e identidade visual para oferecer uma experiência prática e intuitiva ao usuário.

---

# 🚀 Funcionalidades

O sistema foi projetado para oferecer:

- 🍽️ Visualização clara e organizada do cardápio
- 🛒 Carrinho de compras dinâmico
- 💳 Múltiplas formas de pagamento (PIX, cartão e dinheiro)
- 📦 Acompanhamento do pedido em tempo real
- 📱 Interface moderna e responsiva
- ⚙️ Painel administrativo para gestão de:
  - pedidos
  - produtos
  - usuários
